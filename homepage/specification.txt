HTML
header：文字やセクションのヘッダーを表すタグ。
div ：　要素を囲ってグループ化するタグ
a : 指定の要素をハイパーリンクとして扱うタグ
h1,h2,h3 : コンテンツの見出しに使うタグ。数字が小さいほど文字が大きくなり強い意味を持つ。
main : 主要なコンテンツを表すタグ。
footer : コンテンツのフッターを表すタグ。著作情報や関連記事へのリンクなどを記述する際によく使われる。
pre : 整形済みテキストを表すタグ。HTMLの改行などはサイトのページに反映されないことが多いが、これを使うとそのまま反映される。
table : テーブル（表）を定義するタグ。
tr : テーブルの行を定義するタグ。
th : テーブルの見出し項目を定義するタグ。


CSS
height : 要素の高さを指定する
width : 要素の幅を指定する。
background-color : 要素の背景色を指定する。
font-saize: 要素の文字の大きさを指定する。
margin-right : 要素の右側に余白を取る。
